# jQuery Github Activity

This plugin renders the activity of github users as a list to a given element. It makes use of the Github API v3.

## Requirements

* jQuery 1.7.2
* jquery.timeago

## Events

### Supported

* CreateEvent
* FollowEvent
* PushEvent
* WatchEvent

### Not yet supported

* CommitCommentEvent
* DeleteEvent
* DownloadEvent
* ForkEvent
* ForkApplyEvent
* GistEvent
* GollumEvent
* IssueCommentEvent
* IssuesEvent
* MemberEvent
* PublicEvent
* PullRequestEvent
* PullRequestReviewCommentEvent
* TeamAddEvent
